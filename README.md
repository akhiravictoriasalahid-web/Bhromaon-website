# 🌍 Bhromaon Travel Website

A responsive travel landing page built from a Figma design using HTML and SCSS.
This project focuses on clean layout, responsiveness, and pixel-perfect UI implementation.

---

## 🚀 Live Demo

👉 https://akhiravictoriasalahid-web.github.io/Bhromaon-website/

---

## ✨ Features

* Fully responsive design (desktop, tablet, mobile)
* Clean and modern UI
* Pixel-perfect conversion from Figma design
* Organized SCSS structure
* Smooth layout using Flexbox

---

## 🛠 Built With

* HTML5
* SCSS (Sass)
* Flexbox

---

## 📚 What I Learned

* Converting Figma design into real code
* Responsive design using media queries
* Layout techniques using Flexbox
* Structuring styles using SCSS
